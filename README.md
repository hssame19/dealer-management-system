#  Sistema de gerenciamento baseado em console, implementado em Java, que modela o controle de estoque de veículos, o gerenciamento de clientes e o processamento de vendas sob regras de negócio definidas.

# Funcionalidades
- Gerenciamento de estoque de veículos
- Cadastro e gerenciamento de clientes
- Processamento de transações de venda
- Validação de regras de negócio (disponibilidade, preços, restrições)
- Interface via linha de comando
- 

# Detalhes técnicos
- Linguagem: Java
- Interface: Linha de comando (CLI)
- Abordagem: OOP
